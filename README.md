# Web Clipper

A Chrome extension to capture, highlight, and share web content.

## Features

- **Two modes**: Remove (start full, exclude items) or Add (start empty, include items)
- **Text highlighting** with 6 colors
- **Preview** before exporting
- **Export as**: Markdown text, image, or share directly
- **Logseq/Obsidian compatible**: `==highlighted text== (Color)`

## Usage

1. Click the extension icon
2. Choose **Remove** or **Add** mode
3. Click elements to toggle selection
4. Highlight text with colors (optional)
5. Click **Capture** → Preview → Export

## Installation

1. Clone this repository
2. Go to `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked** and select the folder
